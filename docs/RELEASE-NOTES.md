# Shiftframe release notes

## 1.0.3 — schema validation hotfix

- Corrected the Brand Canvas desktop-height range to use a Shopify-valid `20px` increment across `520–900px`.
- Added automated validation for range-step divisibility in theme settings.
- No storefront feature, layout, preset, or merchant-data migration is required.

## 1.0.2 — documentation and support release

- Published public merchant and developer documentation.
- Added installation, customization, commerce, troubleshooting, FAQ, accessibility, and support-policy pages.
- Added a public structured support form and automatic acknowledgement workflow.
- Replaced placeholder documentation and support URLs in theme metadata.
- Preserved all storefront functionality from 1.0.1.

## 1.0.1 — Theme Store requirements release

- Added complete `/listings` install states for Shiftframe, Chromalab, Pantryprint, and Quietform.
- Added the required Shiftframe preset and theme support metadata.
- Corrected Liquid parser errors in featured collection, Image Reveal, collection, search, Product Atlas, and structured data.
- Corrected the product block name-length issue.
- Hardened requirements validation, defaults, accessibility, responsive media, and release packaging.

## 1.0.0 — initial Shiftframe release

- Introduced the original Shiftframe visual system, four presets, native SVG motifs, custom icons, signature commerce sections, alternate templates, product system, cart drawer, search, localization, customer accounts, and complete Shopify theme architecture.
