# Products, commerce, discovery, and localization

## Product setup

For every important product, review the title, vendor, description, media, alternative text, focal points, variants, native swatches, SKU, inventory, unit pricing, compare-at pricing, quantity rules, volume pricing, selling plans, pickup availability, recommendations, complementary products, and SEO fields.

Shiftframe supports images, Shopify-hosted video, external video, 3D models, variant images, linked-product option values, dynamic checkout, Shop Pay installment messaging, gift-card recipient fields, app blocks, and Custom Liquid.

## Variant behavior

Variant selectors update availability, product ID, price, compare-at price, unit price, media, inventory messaging, SKU, URL state, pickup availability, and purchase-button state. Test at least one available, sold-out, and unavailable combination before launch.

## Bundle Studio

Bundle Studio adds selected available variants as separate native Shopify cart lines in one request. It does not create a fixed-price Shopify bundle by itself and it does not simulate automatic discounts. Configure discounts in Shopify or use a compatible bundling app when fixed pricing, component inventory, or subscription bundles are required.

## Cart

The theme includes an AJAX cart drawer and a full cart page. Test line updates, removal, discounts, selling-plan labels, unit prices, cart notes, shipping-progress messaging, accelerated checkout, and checkout handoff.

Any free-shipping threshold shown by the theme must match the store's real shipping configuration.

## Search and filtering

Configure collection and search filters in Shopify Search & Discovery. Keep product types, vendors, tags, option values, and metafield definitions consistent so filters remain useful.

Test predictive search, full search, product/article/page results, sorting, pagination, active filters, no-result behavior, and browser Back/Forward navigation.

## Recommendations

Related recommendations are generated by Shopify. Complementary products are configured through Shopify Search & Discovery. Results can vary by catalog data and Shopify's recommendation engine.

## Markets and languages

Enable the header/footer country and language selectors when the store has multiple markets or languages. Verify currency display, translated theme strings, translated merchant content, long labels, and right-to-left presentation where relevant.

## Customer accounts

Shiftframe includes login, registration, activation, password reset, account, order, and address templates. Test the account mode enabled on the actual store because customer-account behavior depends on Shopify configuration.

## Apps

Use app blocks in product information, featured product, and footer areas where supported. After installing an app, test its blocks, app embeds, scripts, accessibility, visual hierarchy, and performance on mobile and desktop.
