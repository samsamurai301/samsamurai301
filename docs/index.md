# Shiftframe theme documentation

**Current release:** 1.0.3  
**Platform:** Shopify Online Store 2.0  
**Support:** [Open the Shiftframe support form](https://github.com/samsamurai301/samsamurai301/issues/new?template=shiftframe-support.yml)

Shiftframe is a brand-first editorial commerce and product-launch theme for focused Shopify catalogs. This documentation covers installation, presets, theme settings, sections, templates, commerce features, accessibility, troubleshooting, updates, and support.

> Keep an unpublished duplicate of the theme before changing code, installing apps, or applying an update.

## Theme presets

| Preset | Direction | Best fit |
|---|---|---|
| **Shiftframe** | Editorial, typographic, publication-inspired | Fashion, fragrance, art editions, publishing |
| **Chromalab** | High-energy, graphic, experimental | Limited drops, youth products, design objects |
| **Pantryprint** | Warm, tactile, label-driven | Food, beverage, wellness, home goods |
| **Quietform** | Restrained, architectural, product-focused | Furniture, jewelry, premium technology |

## Signature capabilities

Shiftframe includes Graphic Hero, Product Atlas, Drop Room, Story Chapters, Bundle Studio, Specimen Grid, Image Reveal, Proof Wall, Brand Canvas, editorial collection templates, launch product templates, a campaign page, a brand-book page, native SVG motifs, original icons, cart drawer, predictive search, localization, filtering, selling plans, pickup availability, rich product media, app blocks, and customer-account templates.

## Getting support

Use the [public Shiftframe support form](https://github.com/samsamurai301/samsamurai301/issues/new?template=shiftframe-support.yml). Include the theme version, store URL, affected page, reproduction steps, expected result, actual result, browser/device details, and screenshots or a short recording.

GitHub support requests are public. Never include passwords, customer information, API keys, collaborator codes, private app credentials, or other sensitive data. Sensitive security reports should be sent privately to [sohangkumar.patel@gmail.com](mailto:sohangkumar.patel@gmail.com?subject=Private%20Shiftframe%20security%20report).

## Documentation status

This documentation is maintained for Shiftframe 1.0.3. Shopify changes its platform over time, so verify platform-specific administration steps against current Shopify documentation when a screen or label has changed.
